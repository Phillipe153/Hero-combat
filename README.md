O projeto "Combate de Heróis - Super Trunfo" é uma aplicação web desenvolvida utilizando React, fazendo uso intensivo da Context API para gerenciamento global de estado. A proposta é oferecer uma experiência interativa e divertida, onde os usuários podem escolher seus heróis favoritos e colocá-los em batalhas estilo "Super Trunfo".

Tecnologias Utilizadas:

React: A escolha do React como framework principal permite a construção de interfaces de usuário dinâmicas e reativas, facilitando a atualização instantânea da interface conforme as ações do usuário.

Context API: A Context API do React será empregada para gerenciar o estado global da aplicação, permitindo o compartilhamento eficiente de dados entre componentes sem a necessidade de prop drilling.

Material-UI: A biblioteca Material-UI será utilizada para garantir uma interface de usuário moderna e atraente. Seus componentes prontos para uso aceleram o desenvolvimento e fornecem uma estética consistente.

Tailwind CSS: O Tailwind CSS será empregado para a estilização da aplicação, fornecendo uma abordagem utility-first que simplifica a criação e personalização de estilos de forma eficiente.

Funcionalidades Principais:

Seleção de Heróis: Os usuários poderão escolher seus heróis favoritos a partir de uma lista pré-definida, incluindo informações como nome, poderes, e pontuações.

Batalhas de Super Trunfo: Os heróis selecionados serão colocados em batalhas entre si, onde diferentes atributos (como força, velocidade, inteligência, etc.) serão comparados. O herói maior somatório de powerstats ganha a rodada.

Contexto Global: A Context API do React será usada para gerenciar o estado global da aplicação, garantindo que informações importantes estejam disponíveis em toda a aplicação de forma eficiente.

Interface Responsiva: A aplicação será desenvolvida com uma abordagem de design responsivo, garantindo uma experiência de usuário consistente em dispositivos desktop e móveis.

Objetivos do Projeto:

Criar uma experiência de usuário envolvente e divertida.
Explorar e implementar o gerenciamento de estado global com a Context API.
Integrar componentes visuais do Material-UI para uma interface atrativa.
Utilizar o Tailwind CSS para uma estilização eficiente e flexível.
O "Combate de Heróis - Super Trunfo" visa proporcionar não apenas uma experiência lúdica, mas também servir como um exemplo prático de como integrar tecnologias populares do ecossistema React para desenvolver uma aplicação interativa e visualmente atraente.
